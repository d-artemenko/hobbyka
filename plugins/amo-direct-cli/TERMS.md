# Terms

`amo-direct-cli` is provided as-is for authorized read-only access to amoCRM accounts. Use it only with an account and token you are permitted to access.

The user is responsible for granting minimum necessary amoCRM permissions, protecting credentials, reviewing filters and results, and complying with applicable company policies and law. The CLI does not expand the permissions of the configured token.

Do not add write operations or disable sensitive-data safeguards without a separate security review.
